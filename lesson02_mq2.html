01

값 한 번 읽기 — 직접 타이핑

가장 단순한 시작. ADC 핀에서 값을 한 번 읽고 출력합니다.

python · 01_read_once.py 
from machine import ADC, Pin

gas_sensor = ADC(Pin(26))
print(gas_sensor.read_u16())

실행하면 0 ~ 65535 사이의 숫자 하나가 출력됩니다.

📖 ADC가 뭐예요?

ADC는 Analog-to-Digital Converter — "아날로그를 디지털로 바꾸는 변환기"의 줄임말이에요.

센서가 만들어내는 신호는 아날로그(연속된 전압)지만, 컴퓨터는 숫자(디지털)만 이해할 수 있어요. ADC가 그 사이를 통역해주는 거죠. MQ-2 센서가 측정한 전압을 0~65535 사이 숫자로 바꿔서 우리에게 전달해줘요.

Pico의 26번 핀(GP26)이 ADC0이라는 ADC 입구 역할을 하고 있어서, 그 핀에 연결한 센서값을 읽을 수 있는 거예요.

02

계속 읽기 — 직접 타이핑

python · 02_read_loop.py 
from machine import ADC, Pin
import time

gas_sensor = ADC(Pin(26))

while True:
    print(gas_sensor.read_u16())
    time.sleep(0.5)

while True로 반복, time.sleep(0.5)로 0.5초마다 출력.

03

전압과 % 변환 — 직접 타이핑

python · 03_convert.py 
from machine import ADC, Pin
import time

gas_sensor = ADC(Pin(26))

while True:
    value = gas_sensor.read_u16()
    voltage = value / 65535 * 3.3
    percent = value / 65535 * 100
    print(f"ADC: {value:5d}  |  {voltage:.2f}V  |  {percent:.1f}%")
    time.sleep(0.5)

왜 이렇게 계산하나?

  • ADC는 0 ~ 65535 범위로 값을 표현 (16비트)
  • 0이면 0V, 65535이면 3.3V (Pico의 기준 전압)
  • 같은 데이터를 ADC 원본 · 전압 · % 세 가지로 표현해 보기

import network
import socket
import json
from machine import ADC, Pin
import time
from wifi_config import WIFI_SSID, WIFI_PASSWORD

gas_sensor = ADC(Pin(26))


def connect_wifi():
    wlan = network.WLAN(network.STA_IF)
    wlan.active(True)
    wlan.connect(WIFI_SSID, WIFI_PASSWORD)
    print("Wi-Fi 연결 중", end="")
    timeout = 20
    while not wlan.isconnected() and timeout > 0:
        print(".", end="")
        time.sleep(0.5)
        timeout -= 1
    if wlan.isconnected():
        ip = wlan.ifconfig()[0]
        print(f"\n✅ http://{ip}")
        return ip
    print("\n❌ Wi-Fi 연결 실패")
    return None

def send_response(conn, content_type, body):
    header = (
        "HTTP/1.1 200 OK\r\n"
        f"Content-Type: {content_type}\r\n"
        f"Content-Length: {len(body)}\r\n"
        "Connection: close\r\n"
        "\r\n"
    ).encode()
    conn.sendall(header + body)


def run_server():
    addr = socket.getaddrinfo("0.0.0.0", 80)[0][-1]
    srv = socket.socket()
    srv.setsockopt(socket.SOL_SOCKET, socket.SO_REUSEADDR, 1)
    srv.bind(addr)
    srv.listen(3)
    print("서버 대기 중...")

    while True:
        conn, _ = srv.accept()
        try:
            req = conn.recv(512).decode("utf-8", "ignore")
            if "GET /data" in req:
                value = gas_sensor.read_u16()
                body = json.dumps({"value": value}).encode()
                send_response(conn, "application/json", body)
            else:
                send_response(conn, "text/html; charset=utf-8", HTML)
        except Exception as e:
            print("오류:", e)
        finally:
            conn.close()

06·2

상태 판정 — SAFE / WARNING / DANGER

무엇이 추가되나?

값에 따라 자동으로 상태를 판정하고, 색이 바뀝니다.

📝 샘플 프롬프트

PROMPT TO AI
정보 카드의 "상태"를 실제 값에 따라 자동으로 바뀌게 해줘. 기준: - 값 < 20000 → SAFE (초록) - 값 < 45000 → WARNING (주황) - 값 ≥ 45000 → DANGER (빨강) 추가로: - 상태에 따라 ADC RAW 카드의 숫자 색도 바꿔줘 - 상태 카드의 테두리 색도 바꿔줘 - 그래프의 선 색도 상태에 따라 바뀌면 좋겠어

from machine import ADC, Pin
import time
from wifi_config import WIFI_SSID as SSID, WIFI_PASSWORD as PASSWORD

gas_sensor = ADC(Pin(26))


def read_average(sensor, count=10):
    total = 0
    for _ in range(count):
        total += sensor.read_u16()
        time.sleep_ms(10)
    return total // count


def connect_wifi():
    wlan = network.WLAN(network.STA_IF)
    wlan.active(True)
    wlan.connect(SSID, PASSWORD)
    print("Wi-Fi 연결 중", end="")
    timeout = 20
    while not wlan.isconnected() and timeout > 0:
        print(".", end="")
        time.sleep(0.5)
        timeout -= 1
    if wlan.isconnected():
        ip = wlan.ifconfig()[0]
        print(f"\n✅ 연결 완료!  →  http://{ip}")
        return ip
    print("\n❌ Wi-Fi 연결 실패")
    return None
